# Reproductor de musica en python

Una aplicacion de escritorio interactiva creada en Python que le permite al usuario reproducir archivos de audio MP3 locales a travez de una interfaz grafica simple.

// Caracteristicas
- Interfaz grafica limpia usando "Tkinter".
- Motor de Reproduccion de audio impulsado por "Pygame".
-Controles completos de reproduccion: Reproducir, Pausar, Reanudar y Detener.
-Filtro automatico para seleccionaer unicamente archivos compatibles.

// Tecnologias Empleadas
- 'Python 3'
- 'Tkinter
- 'Pygame-CE'
